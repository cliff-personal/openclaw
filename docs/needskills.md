# 需要安装的插件列表

## 已安装的自我进化相关插件

### 核心技能

- **self-evolving-skill** (1.0.2) - 自我进化技能
- **recursive-self-improvement** (1.0.0) - 递归自我改进
- **skill-scaffold** (1.0.3) - 技能脚手架，用于创建新技能
- **skill-from-memory** (0.1.1) - 从记忆中生成技能
- **skill-scanner** (0.1.2) - 扫描技能依赖
- **skill-deps** (1.0.0) - 技能依赖管理

### 编程相关

- **opencode-controller** (1.0.0) - OpenCode 编程控制器
- **codex-orchestrator** (1.0.0) - Codex 编排器

### 其他工具

- **gemini** (1.0.0) - Gemini AI 问答
- **session-logs** (1.0.0) - 会话日志分析

## 安装命令

```bash
# 安装 clawhub CLI
npm install -g clawhub

# 安装自我进化相关技能
clawhub install self-evolving-skill
clawhub install recursive-self-improvement
clawhub install skill-scaffold
clawhub install skill-from-memory
clawhub install skill-scanner
clawhub install skill-deps

# 安装编程相关技能
clawhub install opencode-controller
clawhub install codex-orchestrator

# 安装其他工具
clawhub install gemini
clawhub install session-logs
```
